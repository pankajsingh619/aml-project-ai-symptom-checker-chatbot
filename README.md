# AI Symptom Checker Chatbot 🤖🏥

A professional healthcare AI application that helps users understand their symptoms and get health guidance.

## 🚀 Live Demo

**[Try the live app](https://huggingface.co/spaces/pankaj092223/ai-symptom-checker-chatbot)**

## 🎯 Features

- **Emergency Detection** - Identifies critical symptoms
- **Visual Symptom Selector** - 12 symptom buttons with icons
- **Severity Assessment** - Three-level scale (Mild/Moderate/Severe)
- **Possible Causes** - Likelihood-based causes with percentages
- **Recommended Actions** - Personalized health advice
- **Professional Healthcare UI** - Clean, modern design
